# AFKHelper

## Commands

- `/afk`, `/afk on`, `/afk off`, `/afk toggle` control AFK network-only mode.
- `/switch <name>` changes the client session to an offline/cracked account with that name for the next server join. Names must be 3-16 letters, numbers, or underscores.
